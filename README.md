# ポートフォリオサイト

## 概要
Webデザイン・コーディングのポートフォリオサイトです。  
LP制作・バナー制作・Webサイト制作を中心に制作しています。

## サイトURL
https://MM-Design-Lab.github.io/rumi-portfolio/

## 得意なデザイン
やさしく親しみやすい雰囲気のデザインを得意とし、ユーザーにとって「見やすい・伝わりやすい」構成を意識しています。

主に子育て世代など、日常に寄り添ったユーザー向けのデザインを想定していますが、目的やターゲットに応じてトーンや表現を柔軟に調整することが可能です。

## 使用技術
- HTML
- CSS
- JavaScript（学習中）

## 制作で意識していること
- 情報設計（何をどこに置くと伝わりやすいか）
- ユーザー視点での見やすさ・分かりやすさ
- ターゲットに合わせたトーン設計

## 今後の目標
より実務に近い制作物を増やし、UI設計・LP改善などのスキルも高めていきます。
